# Mentions légales

<p style="color: var(--sl-color-text-accent); font-weight: 600;">
  TDS est actuellement une initiative indépendante et communautaire — elle ne constitue pas
  encore un service officiel de l'État du Tchad. Voir la page <a href="/about/mission/">Mission</a>
  pour le détail du statut du projet.
</p>

## Éditeur du site

Ce site de documentation est maintenu par la communauté de contributeurs du projet open source
TDS (Tchad Design System), hébergé sur le dépôt public
[github.com/wargafu/TDS](https://github.com/wargafu/TDS).

## Hébergement

Ce site est hébergé par **GitHub Pages** (GitHub, Inc. — 88 Colin P Kelly Jr St, San Francisco,
CA 94107, États-Unis).

## Propriété intellectuelle

Le code source de TDS (packages, tokens, composants) est publié sous licence
[MIT](https://github.com/wargafu/TDS/blob/main/LICENSE) — libre de réutilisation, y compris à
des fins commerciales, sans obligation d'attribution.

Le contenu éditorial de cette documentation (textes, exemples, illustrations) est publié dans
le même esprit d'ouverture ; contactez les mainteneurs via
[GitHub](https://github.com/wargafu/TDS/discussions) pour toute question de réutilisation.

## Contact

Pour toute question relative à ce site, ouvrez une
[issue GitHub](https://github.com/wargafu/TDS/issues) ou une
[discussion](https://github.com/wargafu/TDS/discussions).
