# Paramètres d'affichage

## Thème clair / sombre

Ce site propose un sélecteur de thème en haut à droite de chaque page (à côté du lien GitHub) :
**Clair**, **Sombre**, ou **Auto** (suit le réglage de votre système d'exploitation). Ce choix
est mémorisé localement dans votre navigateur d'une visite à l'autre.

## Taille du texte

Ce site n'a pas encore de réglage dédié pour la taille du texte. En attendant, utilisez le zoom
de votre navigateur (`Ctrl` + `+` / `Ctrl` + `-`, ou `⌘` + `+` / `⌘` + `-` sur Mac) — l'ensemble
des composants TDS reste lisible et utilisable jusqu'à 200% de zoom, conformément au critère
WCAG 1.4.4 (voir [Principes WCAG](/fondamentaux/wcag/)).

## Langue

Le site est disponible en français. Le support de l'arabe (RTL) est prévu pour les composants
TDS eux-mêmes (voir [RTL & Arabe](/fondamentaux/rtl/)) mais la documentation n'est pas encore
traduite.
