**République du Tchad** s'engage à rendre son site accessible conformément aux principes du
Référentiel Général d'Amélioration de l'Accessibilité (RGAA) et de WCAG 2.1 niveau AA.

Cette déclaration d'accessibilité s'applique à ce site de documentation
(**tds.gouv.td** / **wargafu.github.io/TDS**).

---

## État de conformité

<Aside type="caution" title="Non conforme">
  Ce site est **non conforme** avec le RGAA. Aucun audit d'accessibilité complet
  (manuel, avec tests utilisateurs et technologies d'assistance réelles) n'a été mené à ce
  jour. Nous ne publions pas de taux de conformité tant qu'un audit réel n'a pas été réalisé —
  afficher un chiffre non vérifié serait trompeur.
</Aside>

Ce qui est fait aujourd'hui :
- Les composants TDS sont conçus selon les principes WCAG 2.1 AA (voir
  [Principes WCAG](/fondamentaux/wcag/))
- Des vérifications automatisées (axe-core) tournent en intégration continue sur chaque
  changement, contre le terrain de jeu des composants et les pages de documentation
- Contraste, structure de titres, labels de formulaire, navigation clavier de base sont vérifiés
  automatiquement

Ce qui reste à faire (voir la [roadmap](/communaute/roadmap/)) :
- Audit RGAA/WCAG 2.1 AA complet mené par un auditeur indépendant
- Tests avec de vrais utilisateurs de technologies d'assistance (lecteurs d'écran, navigation
  clavier exclusive, zoom, contraste élevé)
- Publication d'un taux de conformité et d'un plan d'action correctif

---

## Établissement de cette déclaration

Cette déclaration a été établie le 2 juillet 2026 sur la base d'une auto-évaluation partielle
(vérifications automatisées uniquement — pas d'audit manuel).

---

## Signaler un problème d'accessibilité

Si vous rencontrez un obstacle à l'accessibilité sur ce site, merci de le signaler via
[une issue GitHub](https://github.com/wargafu/TDS/issues) en décrivant le problème rencontré
et, si possible, la technologie d'assistance utilisée.
