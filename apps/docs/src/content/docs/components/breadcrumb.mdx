Le fil d'Ariane situe l'utilisateur dans l'arborescence du site. Les séparateurs sont générés en
CSS (`::before`), pas insérés dans le HTML — le contenu lu par un lecteur d'écran reste propre.

<TabItem label="Design">

Structure : liste ordonnée `<ol>` d'items `tds-breadcrumb__item`, séparateur `/` généré en CSS
(`::before`), dernier item en texte simple (`tds-breadcrumb__current`).

</TabItem>
<TabItem label="Code">

### Importation

```css
@import "@tds-tchad/core/components/breadcrumb/breadcrumb.css";
```

```typescript
import { breadcrumb } from '@tds-tchad/core/components/breadcrumb';
```

### Exemple

```html
<nav class="tds-breadcrumb" aria-label="Fil d'Ariane">
  <ol class="tds-breadcrumb__list">
    <li class="tds-breadcrumb__item">
      <a href="/" class="tds-breadcrumb__link">Accueil</a>
    </li>
    <li class="tds-breadcrumb__item">
      <a href="/demarches" class="tds-breadcrumb__link">Démarches</a>
    </li>
    <li class="tds-breadcrumb__item">
      <span class="tds-breadcrumb__current" aria-current="page">Renouvellement de carte</span>
    </li>
  </ol>
</nav>
```

</TabItem>
<TabItem label="Accessibilité">

<Aside type="tip" title="Règles critiques">
  - Utiliser une liste ordonnée `<ol>` — l'ordre est sémantiquement significatif
  - Le dernier élément (page courante) est un texte simple avec `aria-current="page"`, **pas un lien**
  - Le `<nav>` doit avoir un `aria-label="Fil d'Ariane"` (ou équivalent) pour se distinguer de la navigation principale
</Aside>
