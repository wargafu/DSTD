# Badge

<Tabs>
<TabItem label="Présentation">

Les badges indiquent un statut, une catégorie ou un compteur. Ils sont toujours accompagnés
d'un texte visible ou d'un `aria-label` pour l'accessibilité.

## Importation

```css
@import "@tds-tchad/core/components/badge/badge.css";
```

### Variantes

```html
<span class="tds-badge tds-badge--default">Par défaut</span>
<span class="tds-badge tds-badge--info">Information</span>
<span class="tds-badge tds-badge--success">Approuvé</span>
<span class="tds-badge tds-badge--warning">En attente</span>
<span class="tds-badge tds-badge--danger">Rejeté</span>
```

### Avec point de statut

```html
<span class="tds-badge tds-badge--success">
  <span class="tds-badge__dot" aria-hidden="true"></span>
  En ligne
</span>

<span class="tds-badge tds-badge--danger">
  <span class="tds-badge__dot" aria-hidden="true"></span>
  Hors ligne
</span>
```

### Forme rectangulaire

```html
<!-- Modificateur --square pour les catégories et tags -->
<span class="tds-badge tds-badge--info tds-badge--square">Document</span>
<span class="tds-badge tds-badge--default tds-badge--square">Formulaire</span>
```

### Usage gouvernemental

```html
<!-- Statut d'un dossier -->
<table class="tds-table">
  <tbody>
    <tr>
      <td>Demande de CNI</td>
      <td><span class="tds-badge tds-badge--warning">En cours</span></td>
    </tr>
    <tr>
      <td>Certificat de résidence</td>
      <td><span class="tds-badge tds-badge--success">Prêt</span></td>
    </tr>
    <tr>
      <td>Passeport</td>
      <td><span class="tds-badge tds-badge--danger">Rejeté</span></td>
    </tr>
  </tbody>
</table>
```

</TabItem>
<TabItem label="Accessibilité">

<Aside type="tip" title="Le badge ne doit jamais être la seule information">
  Le point de statut (`tds-badge__dot`) est décoratif (`aria-hidden="true"`) — le texte du badge
  porte toujours l'information. N'utilisez jamais une couleur seule (ex. un badge vide, juste
  coloré) pour communiquer un statut : ajoutez systématiquement un libellé texte.
</Aside>
