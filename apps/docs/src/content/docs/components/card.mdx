<TabItem label="Présentation">

Les cards regroupent du contenu lié dans une surface visuellement délimitée.
Elles structurent les portails citoyens, tableaux de bord et listes de services.

| Variante | Classe | Usage |
|---|---|---|
| `flat` (défaut) | `tds-card--flat` | Liste dense, faible hiérarchie visuelle |
| `elevated` | `tds-card--elevated` | Mise en avant, grille de services |
| `outlined` | `tds-card--outlined` | Contexte à fond déjà chargé (évite l'ombre) |

</TabItem>

## Importation

```css
@import "@tds-tchad/core/components/card/card.css";
```

### Structure de base

```html
<div class="tds-card">
  <div class="tds-card__header">
    <h3 class="tds-card__title">Demande de carte d'identité</h3>
  </div>
  <div class="tds-card__body">
    <p class="tds-card__subtitle">Déposée le 15 janvier 2024</p>
    <p class="tds-card__description">
      Votre demande est en cours de traitement par la préfecture de N'Djamena.
    </p>
  </div>
  <div class="tds-card__footer">
    <span class="tds-badge tds-badge--warning">En cours</span>
    <a href="/dossier/001234" class="tds-link tds-link--standalone">Voir le dossier</a>
  </div>
</div>
```

### Card interactive (lien)

```html
<a href="/service/cni" class="tds-card tds-card--elevated">
  <div class="tds-card__body">
    <h3 class="tds-card__title">Carte nationale d'identité</h3>
    <p class="tds-card__description">
      Demandez ou renouvelez votre carte d'identité en ligne.
    </p>
  </div>
  <div class="tds-card__footer">
    <span class="tds-link tds-link--standalone">Faire une demande</span>
  </div>
</a>
```

<TabItem label="Accessibilité">

<Aside type="tip" title="Règles critiques">
  - Une card interactive entière (`<a class="tds-card">`) doit avoir un contenu de lien clair — évitez "Voir plus" isolé sans contexte dans le texte du lien environnant
  - Le titre de card (`tds-card__title`) doit utiliser un niveau de titre cohérent avec la hiérarchie de la page (`h2`/`h3`), jamais choisi uniquement pour son style visuel
  - Une grille de cards doit rester navigable au clavier dans un ordre logique (ordre du DOM = ordre visuel)
</Aside>
