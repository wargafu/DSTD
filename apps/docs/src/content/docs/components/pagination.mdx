<TabItem label="Présentation">

Navigation entre pages de résultats. Les liens précédent/suivant en bordure de liste utilisent
`aria-disabled="true"` plutôt que d'être supprimés, pour garder une structure de liste stable.

## Do / Don't

<div class="tds-do-dont">
  <div class="tds-do-dont__card tds-do-dont__card--do">
    <div class="tds-do-dont__heading">✓ Faire</div>
    <ul>
      <li>Utiliser des liens réels (<code>&lt;a href&gt;</code>) pour que la pagination fonctionne sans JavaScript</li>
      <li>Tronquer avec une ellipse au-delà de 5-7 pages visibles</li>
    </ul>
  </div>
  <div class="tds-do-dont__card tds-do-dont__card--dont">
    <div class="tds-do-dont__heading">✗ Éviter</div>
    <ul>
      <li>Afficher toutes les pages d'une liste de 50+ résultats sans ellipse</li>
      <li>Rendre précédent/suivant invisibles (<code>display: none</code>) en bordure de liste plutôt que désactivés</li>
    </ul>
  </div>
</div>

<TabItem label="Design">

Structure : `<nav aria-label="Pagination">` > `<ul class="tds-pagination__list">` d'items
`tds-pagination__link`, page courante marquée `aria-current="page"`, bornes désactivées via
`aria-disabled="true"`, troncature via `tds-pagination__ellipsis`.

</TabItem>
<TabItem label="Code">

### Importation

```css
@import "@tds-tchad/core/components/pagination/pagination.css";
```

```typescript
import { pagination } from '@tds-tchad/core/components/pagination';
```

### Exemple

```html
<nav aria-label="Pagination">
  <ul class="tds-pagination__list">
    <li>
      <a href="?page=1" class="tds-pagination__link" aria-disabled="true" aria-label="Page précédente">‹</a>
    </li>
    <li><a href="?page=1" class="tds-pagination__link" aria-current="page" aria-label="Page 1">1</a></li>
    <li><a href="?page=2" class="tds-pagination__link" aria-label="Page 2">2</a></li>
    <li><a href="?page=3" class="tds-pagination__link" aria-label="Page 3">3</a></li>
    <li><span class="tds-pagination__ellipsis">…</span></li>
    <li><a href="?page=12" class="tds-pagination__link" aria-label="Page 12">12</a></li>
    <li>
      <a href="?page=2" class="tds-pagination__link" aria-label="Page suivante">›</a>
    </li>
  </ul>
</nav>
```

</TabItem>
<TabItem label="Accessibilité">

<Aside type="tip" title="Règles critiques">
  - Envelopper dans un `<nav aria-label="Pagination">`
  - Chaque lien de page a un `aria-label="Page N"` explicite (le chiffre visuel seul ne suffit pas au contexte)
  - La page courante utilise `aria-current="page"`, jamais uniquement une couleur de fond
  - Précédent/suivant désactivés (première/dernière page) gardent `aria-disabled="true"` — ne pas retirer le `href` ni l'élément, pour ne pas casser la structure de liste annoncée par le lecteur d'écran
</Aside>
