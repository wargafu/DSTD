# Roadmap

TDS suit un versionnement sémantique strict. Cette page reflète l'état d'avancement réel du
projet — pour le détail complet, voir le [README du repo](https://github.com/wargafu/TDS#feuille-de-route).

---

## v0.1 — Fondations ✅

- 7 systèmes de tokens (couleurs, typographie, espacement, radius, shadow, motion, z-index)
- Pipeline de génération de tokens à source unique
- Tests automatisés, stylelint, vérifications d'accessibilité automatisées (axe-core)
- Documentation Astro Starlight

## v0.2 — Composants de navigation ✅

17 composants au total : Button, Input, Alert, Badge, Card, Link, Table, Header, Navigation,
Breadcrumb, Pagination, Modal, Skip Link, Footer, Accordion, Tabs, Tooltip.

## v0.3 — Modèles et gouvernance (en cours)

- [ ] Template portail citoyen
- [ ] Template tableau de bord administratif
- [ ] Bindings frameworks réels (`@tds-tchad/react`, `@tds-tchad/vue`)

## v1.0 — Production ready (à venir)

- [ ] Audit RGAA / WCAG 2.1 AA complet, mené par un auditeur externe
- [ ] Déclaration d'accessibilité publique
- [ ] Système d'icônes SVG
- [ ] Support arabe RTL testé de bout en bout
- [ ] Publication npm effective de `@tds-tchad/core`
- [ ] Kit Figma synchronisé aux tokens

---

## Proposer une évolution

Toute contribution passe par une discussion ouverte avant implémentation pour les changements
significatifs (nouveau composant, modification de token existant). Voir la page
[Contribution](/communaute/contribution/) pour le processus complet.
